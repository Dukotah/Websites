# shared/

Optional space for things you reuse across multiple sites — a logo you've made,
a reusable copy snippet, brand reference, stock photos, or notes.

This folder is **not** imported automatically by any site. Sites are intentionally
self-contained so they can't break each other. If you want to reuse a file,
copy it into the site's own `public/` or `src/` folder.

If you never need this, it's safe to delete.
