# Example — The Corner Cup

A complete, working example site built from `sites/_template` for a fictional
neighborhood café. Use it as a reference for how a finished client site looks.

```bash
cd sites/example-cafe
npm install
npm run dev      # http://localhost:4321
```

All of the café's content (name, hours, menu, brand colors) lives in
`src/config.ts` — open it next to `sites/_template/src/config.ts` to see the
"before and after."
